# UI Methods

JsonUI-Scripting have many methods to complete UI.

## Override Methods

The override methods is in new instanced UI and new instanced Modify.override

### setProperties

-   Param: properties
-   Action: add new properties to UI element
-   Ex:
```JavaScript
import { UI, Vanilla } from "jsonui-scripting";

const ui = UI.panel({
	size: [16, 16],
})
```
