# UI Methods

JsonUI-Scripting have many methods to complete UI.

## Override Methods

The override methods is in new instanced UI and new instanced Modify.override

### setProperties

-   Param: properties
-   Action: add new properties to UI element
-   Ex:
```JavaScript
import { UI, Vanilla, Anchor } from "jsonui-scripting";

const ui = UI.panel({
	size: [16, 16],
});
ui.setProperties({
	layer: 5,
	anchor: Anchor.TopRight,
});
```
