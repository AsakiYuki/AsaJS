# UI Methods

JsonUI-Scripting have many methods to complete UI.

## Override Methods

The override methods is in new instanced UI and new instanced Modify.override

### .setProperties:

-   Param: (properties)
-   Action: set or add new properties to UI element
-   Ex:
```JavaScript
import { UI, Modify, Anchor, BindingType, BindingName } from "jsonui-scripting";

const ui = UI.panel({
	size: [16, 16],
});
ui.setProperties({
	layer: 5,
	anchor: Anchor.TopRight,
});

const nekoUI_BG = Modify.register(".hans_common_files/.hans_animated_background.json", "bg_anim");

nekoUI_BG.override.setProperties({
	layer: -999,
}),
```
```json
{
	"7272_ELEMENT_12": {
		"type": "panel",
		"size": [ 16, 16 ],
		"layer": 5,
		"anchor_from": "top_right",
		"anchor_to": "top_right"
	},
	"namespace": "d5dhririrb28db2"
}
```
```jsonc
// .hans_common_files/.hans_animated_background.json
{
	"bg_anim": {
		"layer": -999
	}
}
```

### .addChild:
-   Gemeric: T extends string or UI,
-   Param: (element?: T, properties?, name?, callback?),
-   Action: Add new child controls to element.
-   Ex:
```JavaScript
ui.addChild(UI.image, {
	texture: "textures/ui/White",
}, "test");

nekoUI_BG.override.addChild(); // Use to override all controls by a empty array
```
```json
{
	"7272_ELEMENT_12": {
		"controls": [
			{
				"test@d5dhririrb28db2.2526_ELEMENT_1": {
					"texture": "textures/ui/White"
				}
			}
		]
	},
	"2526_ELEMENT_1": {
		"type": "image"
	},
	"namespace": "d5dhririrb28db2"
}
```
```json
{
	"bg_anim": { "controls": [] }
}
```

### .addBindings:
-   Param: bindings object or array contain bindings object(s)
-   Action: add new bindings to element.
-   Ex:
```JavaScript
ui.addBindings({
	binding_type: BindingType.View,
	source_control_name: "test_toggle",
	source_property_name: BindingName.ToggleState,
	target_property_name: BindingName.Visible,
});
```
```json
{
	"7272_ELEMENT_12": {
		"bindings": [
			{
				"binding_type": "view",
				"source_control_name": "test_toggle",
				"source_property_name": "#toggle_state",
				"target_property_name": "#visible",
			}
		]
	}
}
```
